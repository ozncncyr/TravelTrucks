
# TravelTrucks

A modern camper and caravan rental platform built with React.

---

## 🚀 Features
- Browse and filter a catalog of campers and caravans
- Camper detail pages with reviews, features, and booking form
- Favorites system (persisted)
- Responsive, modern UI
- Custom scrollable content areas
- Deep linking and smooth navigation
- State-preserving back navigation

## 📂 Project Structure
- `src/components/` — Reusable UI components
- `src/pages/` — Main pages (Home, Catalog, Camper Details, Not Found)
- `src/redux/` — State management (Redux Toolkit)
- `src/assets/` — Images and SVG icons
- `src/index.css` — Global styles
